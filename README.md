# Gardun Studio — Client Sites

This repo hosts:
- `/index.html` → Gardun Studio's own site
- `/_template/` → starter file for every new client build
- `/business-name/` → one folder per client, each fully self-contained, sitting directly in the repo root

Live at: `https://znar206.github.io/gardun-studio/`
(Will become `gardun.net` once the domain is bought — same structure, just repoint DNS.)

---

## Adding a new client

1. In the repo root, click "Add file" → "Create new file"
2. Type the filename as: `business-name/index.html` (lowercase, hyphens only, no spaces — this auto-creates the folder)
3. Paste their code, commit
4. Go into that new folder, upload any images (banner, logo, etc.) the same way
5. Test by visiting `znar206.github.io/gardun-studio/business-name/`
6. Generate their QR code pointing to that exact URL
7. Log it in the table below

## Editing an existing client's site

1. Find their folder name in the table below
2. Edit only files inside `business-name/`
3. Commit — changes go live in 1-2 minutes

## Naming rules (keep these consistent)

- Lowercase only
- Hyphens instead of spaces (`nay-cosmetic`, not `Nay Cosmetic`)
- Business name only, no extra words
